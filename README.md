## PARK HERE 주요 기능 정리

### 현 위치 기반 주차 추천 리스트

- 기존 서비스: 위치/시간 입력 → 지도 탐색 → 리스트 확인 → 차량번호 입력 → 결제
- 우리 서비스: <strong>앱 실행 시 즉시</strong> 현 위치 주변 주차 가능 공간 자동 추천 → 주차 시간 선택 → NFC 결제
- 최소 1~2단계 절차 단축, <h3>즉시성 확보</h3>



### 곧 나감 기능 (회전율 확보 핵심)

- 기존 문제:
    - 빈자리 표시되어도 이미 점유
    - 무단 점유/시간 초과 잦음
- 우리 기능:
    - 이전 이용자가 마감 시 <h3>“곧 나감” 버튼</h3> 클릭 → 곧 비는 자리로 표시
    - 주변 사용자: <h3>예측 가능한 빈자리 확보</h3>
    - 이용자: 마감 인지 효과 → 무단 점유 방지
- “빈자리 예측 불확실성” 해결, 회전율 증가

---

### NFC 현장 결제

- 기존 문제: 예약 결제 후 자리 없음 → 환불 번거로움, 시간 낭비
- 우리 기능:
    - 예약 기능 <h3>과감히 배제</h3>
    - 현장 도착 → <h3>NFC 태그 1번</h3>으로 결제
- <h3>예약 실패·환불 문제 원천 차단</h3>, 단기·즉시 주차 수요 최적화

---

### AI 주차 예보

- <h3>서울시 실시간 도시 데이터 + Google Maps API</h3>h3> 기반
- 시간·요일·날씨·지역 특성 반영 → <h3>혼잡도(여유/보통/혼잡) 예측</h3>
- 예: 도착 10분 전 “혼잡” 알림 → 경로 재조정
- 장기적으로: <h3>개인 로그 축적 → 초개인화 추천</h3>
- 기존 서비스가 못 푼 <h3>빈자리 예측 불확실성</h3> 보완

### 🔎 AI 주차 예보 동작 원리 정리

<h3>1. 기본 데이터 학습</h3>
    - 서울시 제공 **구역별 혼잡도 데이터** + **주차장 위치 데이터**를 결합해 학습.
    - 시간대별 누적 혼잡 패턴을 기반으로 예측.
<h3>2. 가중치 부여 규칙</h3>
    - <h3>주말</h3>h3>: 전체 혼잡도 1.2배 ↑
    - <h3>회사/업무 구역</h3>: 출근(06~~09시) +1.2배, 퇴근(16:30~~21시) -0.8배
    - <h3>문화/상업 구역(백화점·쇼핑몰 등)</h3>:
        - 평일: 10~~11시, 18~~21시 +1.3배
        - 주말: 10~18시 +1.6배
    - <h3>날씨 보정</h3>:
        - 비·천둥·눈 등 악천후 → 실내 주차장 +1.12배, 실외 주차장 -0.8배
3. <h3>결과 산출 방식</h3>
    - 기본 혼잡도 데이터 × 위 가중치 요소 적용
    - <h3>여유 / 보통 / 혼잡</h3> 각각의 점수를 계산 후, 가장 높은 점수에 해당하는 등급을 결과로 반환
        - 예: 여유 0.9 / 보통 0.5 / 혼잡 0.0 → <h3>“여유롭다”</h3> 반환

### 🔹Git 명령어
